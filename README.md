# Bug Bounty Report: WackoPicko

This repository contains a full security assessment of the vulnerable web application **WackoPicko**.  
The report includes reconnaissance, vulnerability discovery, exploitation, and recommendations.

## Contents
- Full PDF report (`report.pdf`)
- Screenshots of exploitation steps
- Detailed write-ups for each vulnerability

## Vulnerabilities Found
- SQL Injection (Authentication Bypass + Data Extraction)
- Reflected XSS
- Unrestricted File Upload
- IDOR (Insecure Direct Object Reference)

## Tools Used
- Nmap
- Gobuster
- SQLMap
- Burp Suite
- Manual exploitation
